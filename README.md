# alts-private
Шаблон репозитория преподавателя.

## Статичные файлы

`secrets.yaml` - хранилище конфиденциальной информации для тестов.

Формат:
```yaml
<ключ>: <значение>
```

Пример:
```yaml
lab2_var4:
  - input: 2,1,2
    expected_output: 1,2,3
  - input: 1,2,3
    expected_output: 3,2,1
  - input: 4,4,4
    expected_output: 3,2,1
```

---

`students.yaml` - информация о студентах.

Формат:
```yaml
<код>:
    name: <ФИО>
    group: <группа>
    variant: <вариант>
```

Пример:
```yaml
88a3b0f7afb7d18e70842fd5cdd5f49f:
    name: Иванов Иван Иванович
    group: 22-ХХ
    variant: 4
```

## Динамические файлы
`results.csv` - результаты проверки лабораторных работ. Генерируется при выполнении соответствующей задачи.

Формат:
```csv
<код>  <ФИО>  <группа>  <баллы_лр1>  <баллы_лр2>  <баллы_лрX>  <средний_результат>
```

Пример:
```csv
Код  ФИО  Группа  ЛР1  ЛР2  Среднее
88a3b0f7afb7d18e70842fd5cdd5f49f  Иванов Иван Иванович  22-ХХ  80  100  90
```
